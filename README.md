## Abstract

**Background:** Accurate estimation of gestational age and expected date of delivery is essential in obstetric practice. In Nepal, clinical documentation commonly utilizes the Bikram Sambat (B.S.) calendar, creating practical challenges when performing obstetric date calculations manually.

**Objective:** To develop a lightweight, accessible, and clinically practical obstetric calculator capable of determining Estimated Date of Delivery (EDD) and Period of Gestation (POG) using LMP entered in the Nepali (B.S.) date system.

**Methods:** The calculator applies standard obstetric dating principles (Naegeleâ€™s rule: LMP + 280 days) and computes gestational age relative to the current date. The application is built using web technologies (HTML, CSS, JavaScript) and deployed as a Progressive Web App (PWA) to enable offline functionality on Android devices.

**Results:** The tool provides rapid and reliable obstetric date calculations using the Nepali calendar, reducing manual calculation errors and improving efficiency in outpatient, inpatient, and primary care settings.

**Conclusion:** This application offers a practical digital solution for obstetric date calculation within the Nepali healthcare system and may support improved workflow efficiency in clinical practice.

## Background

Accurate determination of gestational age and expected date of delivery is fundamental in obstetric care.  
In Nepal, clinical documentation frequently utilizes the **Bikram Sambat (B.S.) calendar**, creating the need for a reliable obstetric calculator based on the Nepali date system.

This application was developed to provide a simple, fast, and clinically practical solution for healthcare professionals working within the Nepali calendar framework.

## Clinical Functions

The calculator performs:

1. **EDD Calculation**
   - Based on Naegeleâ€™s rule  
   - EDD = LMP + 280 days (40 weeks)

2. **Gestational Age (POG) Calculation**
   - Computes current gestational age in weeks (and days, if applicable)
   - Automatically calculates relative to the current date

All calculations are performed using **Nepali (B.S.) date input**.
