## Overview

Obstetrics Calculator is a simple web application developed to calculate:

- **Estimated Date of Delivery (EDD)**
- **Period of Gestation (POG) / Gestational Age**

using **Last Menstrual Period (LMP)** in **Nepali (Bikram Sambat ‚Äì B.S.) date**.

This tool is designed for healthcare professionals in Nepal who use the Nepali calendar system in clinical practice.

---

## Features

- Calculate EDD (LMP + 280 days)
- Calculate current POG (weeks)
- Nepali (B.S.) date input
- Simple and mobile-friendly interface
- Works offline on Android (PWA support)
- No login required
